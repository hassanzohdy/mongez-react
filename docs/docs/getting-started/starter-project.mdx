title: Mongez Starter Project
---

If you're going to start a new react app, it's recommended to clone [Mongez React Starter Project](https://github.com/hassanzohdy/mongez-react-start-project).

This will provide you a configured and ready-to-use app as it will defines the following:

- [Dot Env File](./../project-structure/dot-env)
- Admin Panel App
    - With **Administrators/Administrators** Groups pages.
    - **Login Page**.
    - Full Setup for [Admin Dashboard In Mongez](./../admin/introduction).
    - Settings Page for app general settings.
    - And much more.
- Front office app which covers the poject itself.
- Shared directory with:
    - [Config File](./../project-structure/shared/config)
    - [Modules List File](./../project-structure/shared/modules)
    - [Shared Provider File](./../project-structure/shared/shared-provider)
    - **locales** directory with most common keywords used in any projects, in `en` and `ar` locale codes.
    - **assets** directory to add all of your static images/videos...etc.
