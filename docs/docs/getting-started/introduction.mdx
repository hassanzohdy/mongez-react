**Mongez React** is a comperhinsive package that transforms your [CRA App](https://github.com/facebook/create-react-app) 
into a full managed framework.

## An amazing React App Manager is coming to twon

Let's see some but not all of what **Mongez** can do

- ✅ Desiging project structure properly.
- ✅ Encouraging for clean and readable code with `TypeScript`.
- ✅ Powerful [Router](./../router/introduction) to handle routes, middlewares and shared layouts.
- ✅ [Localization](./../localization/introduction) in its simplest form to handle multiple languages.
- ✅ [Customizable Configurations](./../config/introduction) to set your entire app configurations.
- ✅ Powerful [Form Components](./../forms/introduction) with +25 ready made components.
- ✅ A Full Http (using [Axios](https://github.com/axios/axios)) requests manager with ease of use.
- ✅ Creating multiple apps like `admin` and `front-office` in one project with standalone/shared components files.
- ✅ Fully lazy loaded module based for fast loading.
- ✅ Helpful [Layout Components](./../layout/introduction) for wrapping pages components.
- ✅ Tons of more [Helpful Components](./../components/introduction).
- ✅ Full featured and powerful [Admin Panel](./../admin/introduction).
- ✅ Full management of [Meta Data](./../advanced/metadata).
- ✅ [Cache Manager](./../cache/introduction). for better usage with local storage.
- ✅ [Events Handler](./../advanced/events) for listening and triggering events.
- ✅ [Encryption](./../advanced/encryption) for encrypting and decrypting data.
- ✅ A [wizard](./../advanced/wizard) to generate modules for your admin panel and front office.
- ✅ Helpful [hooks](./../hooks/introduction) for your components.
